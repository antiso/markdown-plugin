# Markdown Editor Plugin for Eclipse

This plugin is used with Eclipse Indigo

Markdown parser [markdown4j][m4j] is used.

**Installation Instruction**
* Download the [plugin][] jar
* Copy it to the *dropins* folder of eclipse.
* Restart eclipse
